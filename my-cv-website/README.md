# My CV Website

This project is a personal CV website that showcases my professional experience, education, and skills. It is designed to be visually appealing and easy to navigate.

## Project Structure

```
my-cv-website
├── src
│   ├── index.html       # Main HTML document for the CV website
│   ├── styles
│   │   └── main.css     # CSS styles for the CV website
│   └── scripts
│       └── main.js      # JavaScript for interactivity
├── package.json         # npm configuration file
└── README.md            # Project documentation
```

## Getting Started

To set up and run the CV website locally, follow these steps:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/yourusername/my-cv-website.git
   cd my-cv-website
   ```

2. **Install dependencies:**
   If you have npm installed, run:
   ```bash
   npm install
   ```

3. **Open the website:**
   You can open `src/index.html` in your web browser to view the CV website.

## Features

- **Responsive Design:** The website is designed to be responsive and looks great on both desktop and mobile devices.
- **Interactive Elements:** JavaScript is used to add interactivity, enhancing user experience.
- **Custom Styles:** The CSS file contains custom styles to improve the visual appeal of the CV.
